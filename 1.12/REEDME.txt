How to get forge 1.12.2 to load in eclipse

01. get the version that is listed when this file is updated (forge-1.12.2-14.23.5.2814)
02. open the cmd and go to the directory that you want the mod enviorment in
03. drop the forge mdk & the stuff from the respostory in there
04. run "gradlew setupDecompWorkspace"
05. wait until done
06. run "gradlew eclipse"
07. wait until done
08. open eclipse select the folder in you mod enviorment call eclipse and click launch

side note if the run button in elcipse does not work use the runClient.bat
